### 1. Restaurer le code (rollback git)

```bash

git reset --hard backup-2026-04-20-prod-v3
```

### 2. Restaurer les configs DB (templates)

Le dump JSON contient `template.config` complet pour chaque slug. Pour rollback un seul slug :

#### Restaurer `.htaccess` uniquement (rapide)

#### Restaurer tout (.htaccess + landings-app + api + downloads/gs-appelant.apk) depuis l'archive

Pas de SCP (Jailkit). Utiliser le pipe SSH base64 :

```powershell
# 1. Encoder l'archive en base64
$bytes = [IO.File]::ReadAllBytes('backups\vps-prod-backup-20260420-164536.tar.gz')
[IO.File]::WriteAllText('restore.b64', [Convert]::ToBase64String($bytes))

# 2. Pipe vers le VPS (decodage cote serveur)
cmd /c "type restore.b64 | ssh -i %USERPROFILE%\.ssh\obrille_temp -o StrictHostKeyChecking=no obrilleshell@vps112526.serveur-vps.net ""base64 -d > /tmp/restore.tar.gz && md5sum /tmp/restore.tar.gz"""

# 3. Verifier le MD5 (doit etre fbc079800f80e399f6407daf0a2c0d1e)
# 4. Extraire dans /web (en place, ecrase l'existant)
ssh -i $env:USERPROFILE\.ssh\obrille_temp -o BatchMode=yes obrilleshell@vps112526.serveur-vps.net 'cd /web && tar -xzf /tmp/restore.tar.gz && rm /tmp/restore.tar.gz && ls -la'

# 5. Nettoyer
Remove-Item restore.b64
```

## Points cles a savoir pour la prod actuelle

1. **Page blanche / spinner infini** : si ca revient, c'est presque toujours le `vite.config.ts`. Ne PAS reactiver `manualChunks` complexe ni `vite-plugin-compression2`. Vite gere tres bien le splitting tout seul via `React.lazy()`.
2. **Compression Apache** : `mod_deflate` uniquement (gzip dynamique). NE PAS pre-compresser en `.br` ou `.gz` car Apache les re-compresse par-dessus -> `ERR_CONTENT_DECODING_FAILED`.
3. **APK appelant** : un seul fichier servi (`gs-appelant.apk`), arm64-v8a release. Suffisant pour ~99% des smartphones modernes en Cote d'Ivoire.
4. **Backend** : reste sur Vercel (`https://gs-pipeline-app-2.vercel.app/api`), pas d'auto-hebergement. Le proxy PHP `/web/api/` ne fait que relayer (same-origin pour eviter CORS/Facebook).
5. **Trafic** : tout le trafic public (8 slugs landings) tape obrille.com OU coachingexpertci.com -> 0 facturation Vercel pour les visiteurs landing.
